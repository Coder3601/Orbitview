SPACE SITUATIONAL AWARENESS

A unified interface for monitoring low-earth objects, atmospheric entry events, and near-earth asteroid trajectories.

MODULE 01 // SSA-TRACK

SATELLITE TRACKING

Monitor the current orbital state of 70+ high-priority spacecraft. Data is processed through NASA SSC (Satellite Situation Center) to provide precision 3D positioning.

  • Real-time Cartesian mapping to CesiumJS 3D Globe
  • Orbital classification: LEO, MEO, and GEO/HEO
  • Interactive data terminal for altitude and lat/lon metrics
  • Historical orbital trails (Path persistence)

MODULE 02 // CNEOS-ENTRY

ATMOSPHERIC EVENTS

Visualizing fireball and bolide data from the Center for Near-Earth Object Studies. This module maps atmospheric impacts by kinetic energy and altitude.

  • Impact energy heat-mapping
  • Velocity at peak brightness tracking
  • Historical entry database via NASA JPL API
  • Geospatial impact distribution analysis
MODULE 03 // NEOWS-CLOSE

ASTEROID PROXIMITY

Near-Earth Object observation system based on the NeoWs (Near Earth Object Web Service). Tracks asteroids approaching earth within a 7-day window.

  • Hazardous classification (PHA Assessment)
  • Relative velocity and Miss-Distance metrics
  • Estimated diameter (min/max meters) computation
  • Approach timestamping and countdown sequences
[ PROXIMITY_SCAN ]
[ SOLAR_ACTIVITY_MONITOR ]
MODULE 04 // DONKI-SOLAR

SPACE WEATHER

Solar flare and heliospheric monitoring via NASA DONKI. Analyzes X-ray activity and coronal mass ejections to predict ionospheric disruption.

  • X-Class and M-Class flare event logging
  • Active Solar Region (AR) identification
  • Real-time peak-time timestamping
  • Impact assessment for satellite communications
